# ZapPay 

This project is a full-stack digital payments application. It features:

- **Frontend:** Vite + React
- **Backend:** Node.js + Express + MongoDB
- **Features:**
  - User authentication (signup/login)
  - Search for users
  - Transfer money between users
  - Clean, modern UI
- **Docker support**

## Getting Started

### Prerequisites
- Node.js (v18+ recommended)
- npm
- MongoDB (local or cloud)
- Docker (optional)

### Setup

#### Frontend
```
npm install
npm run dev
```

#### Backend
```
cd backend
npm install
npm run dev
```

- Configure environment variables in `backend/.env` as needed.

#### Docker (optional)
- Dockerfile and docker-compose.yml will be provided for easy containerization.

## Folder Structure
- `/frontend` (Vite + React app)
- `/backend` (Node.js + Express API)
